## 前台页面功能

### 1. 彩虹炫彩标题

网站标题使用 CSS 渐变动画实现彩虹流动效果：

```css
.site-title {
    background: linear-gradient(
        90deg,
        #ff6b6b, #feca57, #48dbfb, #ff9ff3, #a29bfe, #00d2d3, #ff6b6b
    );
    background-size: 300% auto;
    -webkit-background-clip: text;
    background-clip: text;
    color: transparent;
    animation: titleFlow 4s linear infinite;
}
```

标题内容完全由后台 `config.json` 中的 `title` 字段控制。

### 2. 动态渐变背景

深色背景带有缓慢移动的渐变动效：
- 背景色在 `#0c0d1e`、`#161a3a`、`#1a1040`、`#0c1535` 之间平滑过渡
- 动画周期20秒，循环播放
- 支持替换为自定义背景图片（后台可设置URL）

### 3. 浮动粒子背景

背景层叠浮动光点粒子，营造星空/宇宙感，由 `app.js` 动态生成20个粒子，随机大小、位置、颜色和动画时长。

### 4. Bing/Google 双引擎搜索

搜索框采用 Bing 风格玻璃拟态设计：
- 支持 `Bing` 和 `Google` 两种引擎一键切换
- 切换按钮位于搜索框下方，圆角药丸形
- 按 Enter 键直接执行搜索，跳转新标签页打开结果
- 搜索关键词自动 URL 编码

### 5. 分类独立配色系统

每个分类对应独立的彩虹配色方案（在 `index.php` 的 `$CAT_COLORS` 数组中定义），包含：

| 分类 | 图标 | 配色主色 |
|------|------|---------|
| featured（推荐位）| 🔥 | 红→橙→粉 |
| AI工具 | 🤖 | 青→蓝→绿 |
| 开发工具 | 💻 | 紫→深紫→粉 |
| 实用工具 | 🛠️ | 青绿 |
| 娱乐专区 | 🎮 | 粉→玫红→红 |
| 学习资源 | 📚 | 金→橙→红 |
| 设计素材 | 🎨 | 红→橙红→金 |
| 影视动漫 | 🎬 | 紫→深紫→蓝 |

新分类不在预设表中时，自动使用默认紫色系。

分类图标优先级：`cat_icons.json`（后台自定义）> 颜色表预设 > 默认📌

### 6. 玻璃拟态卡片

导航卡片采用毛玻璃效果：
```css
.card {
    background: rgba(255, 255, 255, 0.04);
    backdrop-filter: blur(14px);
    border: 1px solid rgba(255, 255, 255, 0.07);
    border-radius: 16px;
}
```

**悬浮交互动效：**
- `translateY(-8px) scale(1.03)` — 卡片上浮放大
- 彩虹顶部边框线条显现（3px渐变条）
- 背景亮度提升，发光阴影增强

### 7. 推荐位 Featured

金色HOT角标 + 更醒目的金色边框，专用于置顶重要链接，与普通分类使用不同的强调样式。

### 8. 点击量统计

每个链接下方显示累计点击次数（`x 次`），通过 `click.php` 中间页跳转时自动 `+1` 更新。

### 9. 底部页脚（4行HTML）

页脚支持完全自定义的4行内容，每行均可写HTML：

| 行号 | 用途 | 默认值 |
|------|------|--------|
| 第1行 | 版权信息 | `© 2026 网站标题` |
| ICP备案号 | 备案号文字 | 支持 `<a>` 标签可跳转 |
| 第3行 | Powered信息 | `Powered with ♥` |
| 第4行 | 额外自定义行 | 留空（可加友链等） |

备案号字段支持两种填写方式：
- 纯文本 → 显示为明亮蓝色文字
- 完整 `<a>` 标签 → 原样输出，可链接至工信部

### 10. 分类ID排序

前台分类显示顺序由 `cat_sort.json` 中的数字ID控制：
- 数字越小越靠前
- `featured` 固定为 ID=0，始终排第一
- 新分类默认分配为当前最大ID+1

### 11. 静态页面支持

生成 `index.html` 后可直接作为静态站点使用，PHP 仅在内容变更时运行一次，适合内容变动不频繁的站点，大幅降低服务器资源消耗。

## 数据存储结构

### data.json — 导航数据

```json
{
  "featured": [
    { "name": "推荐位示例", "url": "https://example.com", "click": 0 }
  ],
  "AI工具": [
    { "name": "ChatGPT", "url": "https://chat.openai.com", "click": 0 }
  ],
  "开发工具": [
    { "name": "GitHub", "url": "https://github.com", "click": 0 }
  ]
}
```

- **featured**：系统推荐位分类（保留，不可删除）
- **每个分类名** 为JSON Key，值为链接数组
- **每个链接**：name（名称）、url（地址）、click（点击数）

### config.json — 站点配置

```json
{
  "title": "个人在线网址合集",
  "subtitle": "✦ 精选网址导航 · 一站直达 ✦",
  "keywords": "导航,AI工具,二次元,动漫,网址",
  "background": "",
  "icp": "京ICP备XXXXXXXX号",
  "icp_url": "",
  "footer_line1": "",
  "footer_line3": "",
  "footer_line4": "",
  "custom_html": "",
  "custom_css": ""
}
```

### cat_icons.json — 分类图标

```json
{
  "AI工具": "🤖",
  "开发工具": "💻"
}
```

覆盖 `index.php` 中预设的分类图标，支持emoji/图片URL/FontAwesome类名。

### cat_sort.json — 分类排序

```json
{
  "featured": 0,
  "AI工具": 1,
  "开发工具": 2
}
```

数值越小排序越靠前，`featured` 固定为0。

## 后台操作指南

### 首次登录

1. 访问 `admin.php`（如 `https://yourdomain.com/admin.php`）
2. 默认密码：`admin123`
3. 登录后立即弹出绿色提示「✅ 登录成功！欢迎回来」
4. 忘记密码？点击登录页「🔑 忘记密码？」查看详细重置步骤

### 修改网站标题

1. 进入后台 → 「📡 站点基本设置」
2. 修改「网站标题」字段
3. 点击「💾 保存站点设置」→ 弹出绿色提示
4. 返回前台刷新页面即可看到变化

### 添加新分类

1. 进入后台 → 「📂 导航分类管理」
2. 填写分类名称（如「游戏资源」）
3. 填写排序ID（可选，不填则自动分配）
4. 点击图标框 → 在素材库中选择图标
5. 点击「+ 新增」→ 弹出绿色提示
6. 新分类出现在表格最底部（按ID排序）

### 修改分类图标/名称/ID

1. 在分类表格中找到目标分类
2. 点击右侧「✏️」按钮
3. 在弹窗中修改：名称 / ID / 图标
4. 点击「确定保存」→ 弹出绿色提示

### 添加导航链接

1. 进入后台 → 「➕ 添加导航链接」
2. 填写名称、网址，选择所属分类
3. 点击「添加」→ 弹出绿色提示「✅ 导航链接「xxx」添加成功！」
4. 新链接立即出现在前台对应分类下

### 编辑链接

1. 进入后台 → 「🗑️ 链接管理」
2. 找到目标链接，点击右侧「✏️」按钮
3. 在弹窗中修改：名称 / 网址 / 点击量
4. 点击「保存修改」→ 弹出绿色提示「✅ 链接「xxx」修改成功！」

### 生成静态首页

1. 进入后台 → 点击顶部「📄 生成静态首页」
2. 弹出确认对话框，点击确定
3. 系统将首页渲染为 `index.html` 文件
4. 弹出绿色提示「📄 静态首页已生成！文件大小：xxx 字节」
5. 用户可直接访问 `index.html` 作为站点入口

### 统计点击量

- 链接被点击后，通过 `click.php` 中间页跳转
- 点击量 `+1` 后跳转到目标网址
- 后台链接列表中可查看每条链接的累计点击次数

## 特色功能详解

### Toast 消息提示系统

后台所有操作完成后都会在页面顶部弹出提示：

| 颜色 | 类型 | 用途 |
|------|------|------|
| 🟢 绿 | success | 保存成功、添加成功、登录成功、链接修改成功 |
| 🔴 红 | error | 密码错误、字段为空、名称重复、权限错误 |
| 🔵 蓝 | info | 普通提示信息 |

所有Toast **3秒自动消失**，多条消息按间隔650ms依次弹出。

### 分类ID排序机制

- `cat_sort.json` 存储每个分类的排序ID（数字）
- 前台/后台均按 `usort()` 函数以ID升序排列
- ID越小越靠前，featured 固定=0 排第一
- 新分类默认分配当前最大ID+1
- 编辑弹窗可随时修改排序ID

### 安全防护

- **XSS防护**：所有用户输入通过 `htmlspecialchars()` 转义后输出
- **数据校验**：添加链接/分类时校验必填字段，拒绝空值
- **删除确认**：所有删除操作均有 `confirm()` 二次确认
- **会话管理**：登录状态通过 `$_SESSION['admin_ok']` 管理
- **URL跳转记录**：`click.php` 中间页隔离跳转，防止外链直接统计
- **属性传参安全**：编辑按钮使用 `data-*` 属性传递参数，避免 emoji 破坏 JS 语法

### 弹窗交互

- ✏️ **编辑分类弹窗**：点击外部遮罩层或按 ESC 键关闭
- ✏️ **编辑链接弹窗**：点击外部遮罩层或按 ESC 键关闭
- 🔑 **忘记密码弹窗**：遮罩层点击关闭，ESC键关闭
- 🚪 **退出登录**：红色按钮，点击后 `confirm()` 二次确认再退出

## 部署说明

### 环境要求

- **PHP 5.6+**（推荐 PHP 7.x 或 8.x）
- 无需数据库，所有数据存储于 JSON 文件
- 无需 Composer 或第三方依赖

### 部署步骤

1. 将所有文件上传至网站根目录（或任意子目录）
2. 确保以下文件有写入权限（PHP需能读写）：
   - `data.json`
   - `config.json`
   - `cat_icons.json`
   - `cat_sort.json`
3. 访问 `admin.php` 登录后台（默认密码：`admin123`）
4. 修改后台密码：编辑 `admin.php` 第29行 `$ADMIN_PASSWORD = 'admin123';`

### 目录权限

```bash
# Linux 服务器示例
chmod 644 data.json config.json cat_icons.json cat_sort.json
chmod 644 *.php *.css *.js
```

### URL重写（可选）

如需将后台入口改为更隐蔽的路径，可重命名 `admin.php`：
```
admin.php → manage.php（或任意名称）
```

## 常见问题

### Q: 忘记后台密码怎么办？

访问 `admin.php` 登录页，点击「🔑 忘记密码？」，弹窗会显示详细重置步骤：
1. 通过 FTP 或文件管理器打开 `admin.php`
2. 找到第29行：`$ADMIN_PASSWORD = 'admin123';`
3. 将 `admin123` 改为你想要的新密码
4. 保存文件后重新登录

### Q: 如何修改后台密码？

直接编辑 `admin.php`，找到第29行：
```php
$ADMIN_PASSWORD = '你的新密码';
```

### Q: 新增的分类没有显示在前台？

新增分类后确保填写了「排序ID」，前台按ID从小到大排序显示。如果ID留空则默认为999（排最后）。

### Q: 图标素材库点击没反应？

确保浏览器支持 JavaScript。若仍有问题，可直接在图标输入框中手动输入emoji。

### Q: 链接点击量不增加？

检查 `click.php` 文件是否存在，且 `data.json` 有写入权限。点击量统计依赖PHP文件写入功能。

### Q: 背景图片不显示？

确认 `config.json` 中的 `background` 字段填写的是完整的图片URL（如 `https://.../bg.jpg`），且图片可公开访问。

### Q: 如何添加推荐位链接？

在后台「➕ 添加导航链接」中，将所属分类选择为 `featured`（推荐位），该链接即可出现在首页顶部的金色推荐位区域。

### Q: 生成静态首页后，如何让访客访问静态版本？

生成 `index.html` 后：
- 如果服务器默认优先读取 `index.html`，访客访问域名时会自动加载静态版本
- 如需强制使用静态版本，可将 `index.php` 重命名（如 `index_dynamic.php`）
- 内容变更后，需重新点击「📄 生成静态首页」更新文件

### Q: 链接编辑弹窗和分类编辑弹窗有何不同？

| | 分类编辑 | 链接编辑 |
|--|---------|---------|
| 入口 | 分类管理表格右侧 ✏️ | 链接管理表格右侧 ✏️ |
| 可修改 | 分类名称 / 图标 / 排序ID | 链接名称 / 网址 / 点击量 |
| 分类名变更 | 会同步修改所有数据文件 | 不涉及分类名修改 |
